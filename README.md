## 📖 Sobre o Projeto

O **VagaLocal** é uma plataforma web desenvolvida durante a graduação em Engenharia de Software na UTFPR.

O objetivo do projeto é oferecer um ambiente simples para divulgação e consulta de vagas de emprego, permitindo que usuários naveguem pelas oportunidades disponíveis através de uma interface intuitiva e responsiva.

---

- CSS3
- JavaScript
